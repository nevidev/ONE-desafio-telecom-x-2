# Desafio realizado
Este projeto utiliza dados previamente tratados de um trabalho anterior para construir e avaliar modelos preditivos de Churn de clientes. Realizamos a padronização, balanceamento de classes com SMOTE e treinamos modelos de Regressão Logística e Random Forest. O objetivo é identificar os fatores mais relevantes para a evasão e aprimorar as estratégias de retenção.
